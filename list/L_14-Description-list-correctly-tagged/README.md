The objective of this technique is to show how to tag a [definition (description) list](https://pdfa.org/glossary-of-accessibility-terminology-in-pdf#definition). 

For such lists, the **[ListNumbering](https://www.pdfa.org/glossary-of-accessibility-terminology-in-pdf/#listnumbering)** attribute is not applicable and therefore should not be set, or set to None.
