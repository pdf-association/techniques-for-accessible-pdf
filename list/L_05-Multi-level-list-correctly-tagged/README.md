The objective of this technique is to show how to tag a multi-level (nested) list. 

The nested list can be understood as a separate list (**L**) contained within a list item (**LI**). The nested list is contained in an **LBody** tag. The containing **LBody** tag can also contain other tags as siblings of the enclosed **L** tag.

Recommended assistive technology [AT](https://pdfa.org/glossary-of-accessibility-terminology-in-pdf/#assitive-technology) behavior: Sub-lists should be exposed when the containing **LBody** is read.

See also hierachical list (L_12), commonly used for taxonomies or organigrams.
